Smart Crop Prediction System

Data-driven crop recommendations for Egypt using satellite & climate data

Crop Recommendation Tool

Select a governorate and month. The system will return crops that match the local conditions based on built-in rules and sample Terra-like monthly data.

How it works (Methods)

  1. System uses monthly averages similar to Terra MODIS (temperature, NDVI, soil moisture).
  2. Built-in crop requirement table (temperature, soil type, water needs).
  3. Rule engine matches region-month conditions to crop requirements and returns best matches.
